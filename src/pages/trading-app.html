AvionFi - Trading App

AvionFi is a full‑stack stock trading application that leverages real-world stock market data from the IEX API. It allows users to manage portfolios, execute trades, and stay updated with live market insights.

Project Context

AvionFi was a duo project with Ty Jacalan. I implemented the portfolio and stock market views, including real-time prices, charts, and news feeds. Ty focused on user authentication, transaction logic, and admin features.

Highlights

  • Live Market Interface
  • User Authentication
  • Admin Console
  • Transaction System

Technologies Used

  • Ruby on Rails
  • Turbo & Stimulus
  • Tailwind CSS
  • Shadcn
  • Chartkick
  • Devise & ActiveMailer
  • IEX Cloud API
