SkillSetGo - Booking App

SkillSetGo is a full-stack appointment scheduling app tailored for freelancers and clients in the Philippines. Built with Ruby on Rails, Tailwind CSS, and Stimulus/Turbo, it enables seamless appointment booking, profile management, payments, and more.

Why I Built This

This project was a team collaboration to create a robust scheduling platform for freelancers. My main focus was backend logic and user experience, particularly user profiles, booking workflows, and services.

Highlights

  • Booking System
  • Payment Integration
  • Notification System
  • User Profile Management
  • Freelancer Service Dashboard
  • Ratings and Reviews
  • User Roles
  • Email Notifications

Technologies Used

  • Ruby on Rails
  • Turbo & Stimulus
  • Tailwind CSS
  • Shadcn
  • Paymongo API
  • Ransack & Kaminari
