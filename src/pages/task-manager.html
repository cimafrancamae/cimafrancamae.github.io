Task Manager

Task Manager is a full-stack task management application built with Ruby on Rails. It features user authentication and CRUD functionality, making it easy to create, organize, and track tasks.

Why I Built This

I created TaskMan to strengthen my understanding of Ruby on Rails, particularly MVC structure, user authentication with Devise, and backend CRUD operations. It helped me explore full-stack web development with clean UI design.

Features

  • User registration and login using Devise
  • Create, update, and delete tasks
  • Prioritize tasks with deadlines
  • Task filtering and searching
  • Responsive design using Tailwind CSS

What I Learned

TaskMan helped me practice building and connecting models, views, and controllers in Rails. I learned how to implement Devise for user authentication and used Tailwind to create a consistent, modern layout.

Technologies Used

  • Ruby on Rails
  • PostgreSQL
  • Devise
  • Tailwind CSS
